Sentiment Analysis of Political News Titles

This is a tool that judges the sentiment of news articles to gain a sense of the popularity of candidates in the 2020 US presidential election. The tool uses various forms of sentiment analysis to determine how a candidate is doing.

Data Sources

CNN, NYTimes and FOX news article titles are used as the text to analyze.

Methodology

Article titles can be found using:

CNN API

NYTimes API

Fox News API

News API
